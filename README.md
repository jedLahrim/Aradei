## Description

[SQFT] Squarefeet | Leasing made easier.

## Installation

```bash
$ pnpm i

$ npx prisma migrate deploy

FRESH DB :
$ npx prisma migrate reset

SEED DB :
$ pnpm ts-node prisma/seed.ts
```

## Running the app

```bash
# production mode
$ pnpm run start:prod

# dev mode
$ pnpm run start:dev

# debug mode
$ pnpm run start:debug
```
